# Role Ansible: hubble_cli

Esta role instala a CLI do Hubble no host remoto, detectando automaticamente a arquitetura e baixando a versão correta.

## Como usar

1. Copie a pasta `roles/hubble_cli` para seu projeto Ansible.
2. Inclua o `playbook_install_hubble_cli.yml` no seu fluxo de execução.
3. Execute normalmente com: `ansible-playbook playbook_install_hubble_cli.yml`
